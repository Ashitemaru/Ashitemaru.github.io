本章主要介绍用言（助动词）应当通过何种活用形式接续指定的用言、体言、助动词及助词。通过本章，读者可以学会如何通过合适的活用形式以连接词语形成词组。

<!-- more -->

# 前向视角的接续规则

接续规则可以从两个视角看。我们先讨论先固定前者，研讨其某种活用形式之后可以接续哪些词语的**前向视角接续**。而固定后者，研讨前者应当使用何种活用形式的**后向视角接续**，我们后续在细致讲解每一个助动词、助词的具体含义的时候给出。

## 动词的前向视角接续

四类动词仅仅是活用规则上有所差距，但实际上在后续接续词语的规则上几乎保持一致，所以我们这里不区分四类动词，仅以「読む」为例词探讨动词各个活用形之后可以接续的词语。

这里用小括号标记的是本手册不深入探讨但会简单讲解的助词，用花括号标记的是本手册不讨论的助词，用星号标记的是可能的争议用法。另外，部分情况下可能出现动词必须先接续指定助动词、助词等后才能接续指定词语的三词接续情况，这里会将这种情况中的中间助动词、助词等用斜体标出。

这里最右边一列表示动词该活用形是否能用于终止句子（即「言い切り」）或者中顿句子（即「中止」）。

另外，为了简化表格，我们将下面的助词简称为**「助詞①」**：

- だけ、くらい、の②、ので、のに、のみ、ほど、まで②、ものか、ものの、ものを、より

将下面的助词简称为**「助詞②」**：

- か①、かしら、こと、さ、ぜ、ぞ、とも、な②、ね、もの、よ、わ、が②、けれど、し、と②、か③、とか、など、なり

将下面的助词简称为**「助詞③」**：

- こそ、すら、しか、さえ、も、や、は

注意，上述三类助词的定义仅仅是为了简化下述表格，并不是严格的语法定义，请不要在本章节之外的场合使用该概念。

{% note info no-icon %}
虽然仅仅是临时定义的概念，但实际上在这个分类下同一类的助词是有一定的内在联系的。

- 「助詞①」的语法作用类似体言，或者直接就是从体言（形式体言）派生得到。
- 「助詞②」基本上为终止助词以及用于终止小句的接续助词，往往标志着小句的结束。
- 「助詞③」中的副助词虽不是格助词，但是在部分条件下具有类似格助词的使用方法。
{% endnote %}

| 活用形式 | 体言 | 用言 | 助動詞 | 助詞 | 言い切り |
| :-: | :-: | :-: | :-: | :-: | :-: |
| 未然形 | | | **読ま**ない | | ― |
| | | | **読ま**れる | | |
| | | | **読ま**せる | | |
| | | | **読ま**ず・**読ま**ん・**読ま**ぬ | | |
| 連用形① | **読み**方 | **読み**取る | **読み**ます | **読み**ながら① | （中止） |
| | | **読み**やすい | **読み**たい | （**読み**っきり） | |
| | | **読み***に*行く | **読み**たがる | （**読み**つつ） | |
| | | | **読み**そうだ② | （**読み**しだい） | |
| | | | | ｛**読み**つ｝ | |
| 連用形② | | | **読ん**だ | **読ん**だり | ― |
| | | | | **読ん**で | |
| | | | | **読ん**でも | |
| | | | | **読ん***だ*きり | |
| | | | | **読ん***だ*ばかり | |
| 連体形 | **読む**こと | | **読む**ようだ | **読む**ばかり | ― |
| | | | | **読む** ＋「助詞①」 | |
| 仮定形 | | | | **読め**ば | ― |
| 終止形 | | | **読む**そうだ① | **読む**な① | （言い切り） |
| | | | **読む**らしい | **読む** ＋「助詞②」 | |
| | | | **読む**べし | \***読む** ＋「助詞③」 | |
| | | | **読む**まい | **読む** ＋「格助詞」 | |
| | | | \***読む**だ | | |
| 命令形 | | | | **読め**よ | （言い切り） |
| 意志形 | | | **読も**う | | ― |

此外还有一些较为特殊的用法列举如下：

- 部分动词的第一连用形可以作为动词的名词形式使用，在「踊りが上手」等句子中充当体言成分。

关于该表的部分补充说明如下：

- 动词使用第一连用形接续体言形成复合体言仅仅适用于部分已经约定俗成的复合用语之中，如「読み方、飲み物、買い物、行き先」等，而使用动词修饰一般体言应当使用连体形。
- 动词使用第一连用形接续「に」后所能接续的动词，仅可以是表现行动的动词，如「行く、来る」等，该用法用于表现这些行动的目的。
- 动词使用连体形除了接续实意体言外，还可以接续含义虚化但暂未归纳为助词的形式体言，有关形式体言的内容我们后续讲解。
- 动词终止形还可以与「にしたがって、に違いない、にしろ」等接续形成惯用语，如「試験が近づくにしたがって」。
- 动词终止形接续格助词这种说法有一定的偏差，更精确的说法应当是小句、词组接续格助词，而终止形在这里起到的作用是标记小句、词组的结束。

类动词助动词的接续方式基本和上表类似，具体差别在后向视角接续章节介绍。

## 形容词的前向视角接续

下述表格与动词的前向视角接续表格格式和约定一致，形容词选用的词例为「高い」。

| 活用形式 | 体言 | 用言 | 助動詞 | 助詞 | 言い切り |
| :-: | :-: | :-: | :-: | :-: | :-: |
| 未然形 | | | **高かろ**う | | ― |
| 連用形① | | **高く**なる | **高く**ない | **高く**て | （中止） |
| | | | | **高く**ても | |
| 連用形② | | | **高かっ**た | | ― |
| 連体形 | **高い**ところ | | **高い**ようだ | **高い** ＋「助詞①」 | ― |
| 仮定形 | | | | **高けれ**ば | ― |
| 終止形 | | | \***高い**だ | **高い**ながら② | （言い切り） |
| | | | \***高い**です | **高い** ＋「助詞②」 | |
| | | | **高い**そうだ① | \***高い** ＋「助詞③」 | |
| | | | **高い**らしい | \***高い** ＋「格助詞」 | |
| 語幹 | **高**台 | **高**望み | **高**そうだ② | | （言い切り） |

关于该表的部分补充说明如下：

- 形容词使用词干接续体言形成复合体言仅仅适用于部分已经约定俗成的复合用语之中，而使用形容词修饰一般体言应当使用连体形。使用词干接续用言类似，修饰一般用言应当使用第一连用形。
- 形容词使用连体形除了接续实意体言外，还可以接续含义虚化但暂未归纳为助词的形式体言，有关形式体言的内容我们后续讲解。
- 形容词终止形接续格助词这种说法有一定的偏差，更精确的说法应当是小句、词组接续格助词，而终止形在这里起到的作用是标记小句、词组的结束。

类形容词助动词的接续方式基本和上表类似，具体差别在后向视角接续章节介绍。

## 形容动词的前向视角接续

下述表格与动词的前向视角接续表格格式和约定一致，形容动词选用的词例为「好きだ」。

| 活用形式 | 体言 | 用言 | 助動詞 | 助詞 | 言い切り |
| :-: | :-: | :-: | :-: | :-: | :-: |
| 未然形 | | | **好きだろ**う | | ― |
| 連用形① | | | **好きで**ない | **好きで**も | （中止） |
| 連用形② | | | **好きだっ**た | | ― |
| 連用形③ | | **好きに**なる | | | ― |
| 連体形 | **好きな**こと | | **好きな**ようだ | **好きな** ＋「助詞①」 | ― |
| 仮定形 | | | | **好きなら**ば | ― |
| 終止形 | | | **好きだ**そうだ① | **好きだ**ながら② | （言い切り） |
| | | | | **好きだ** ＋「助詞②」 | |
| | | | | **好きだ** ＋「助詞③」 | |
| | | | | **好きだ** ＋「格助詞」 | |
| 語幹 | | | **好き**です | **好き**か①③ | （言い切り） |
| | | | **好き**そうだ② | **好き**かしら | |
| | | | **好き**らしい | | |

关于该表的部分补充说明如下：

- 对于形容动词，「助詞②」需要删去「か①③、かしら」。
- 形容动词使用连体形除了接续实意体言外，还可以接续含义虚化但暂未归纳为助词的形式体言，有关形式体言的内容我们后续讲解。
- 形容动词终止形接续格助词这种说法有一定的偏差，更精确的说法应当是小句、词组接续格助词，而终止形在这里起到的作用是标记小句、词组的结束。

类形容动词助动词的接续方式基本和上表类似，具体差别在后向视角接续章节介绍。

## 体言的前向视角接续

体言没有活用的说法，但是其基本上可以通过接续助动词「だ」之后通过该助动词的活用来接续各类词语。

首先我们简单介绍体言与除了助词的词语的接续情况，体言选用的词例为「花」：

| | 体言 | 用言 | 助動詞 | 助詞 |
| :-: | :-: | :-: | :-: |
| 原形 | **花**道 | **花**見る | **花**だ |
| | | | **花**です |
| | | | **花**らしい |
| ＋「の」 | | | **花の**ようだ |
| ＋「だ」 | | | **花だ**そうだ① |

之后介绍体言与助词的接续情况。可以和体言原形直接接续的助词包括：

- 格助词
- だけ、くらい、まで②、ほど、より
- か①、かしら、さ、ぜ、ぞ、ね、\*な②、よ、わ、か③、とか、など、なり
- こそ、すら、しか、さえ、も、や、は

需要先添上「な」之后再接续的助词包括：

- の②、ので、のに、のみ、\*ものか、\*ものの、\*ものを

需要先添上「だ」之后再接续的助词包括：

- \*もの
- が②、けれど、し、\*と②、こと、とも、な②

事实上体言和助词的连用规则较为模糊，本手册这里也只是给出了我们推荐的连用方法。
